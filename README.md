Windows_Service
===============

Windows_Service
1.需求-实现某种功能
2.寻找软件
3.了解软件-好处&弊端
4.安装软件
-注册表
-安装路径
-缓存位置
5.操作软件(基本)
6.设置软件(具体设置)
-主程序的启动方式:自动&手动  msconfig & 辅助软件
-主程序的任务计划(一般为更新):运行&结束&禁用&导出|导入(xml)&删除 plan
-主程序的服务(加载的服务)(后台常驻)::启动&停止|自动&手动&禁止 service
-程序加载(剔除其他AD功能) ren
7.主程序的设置备份
-cmd (copy&7z last-XX.7z date-XX.7z if exist)
8.主程序的更新
-下载
-覆盖升级
-重新安装
9.卸载软件
-主程序的常规卸载
-注册表的清理 辅助软件
-插件 
-主程序进程 task
-服务(系统加载的服务) service
-残余文件(无法删除文件,可能为系统的服务仍在加载这些文件,只要停止服务即可删除!)
